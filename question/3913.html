Как проверять равенство двух float

Проверка равенства двух чисел с плавающей точкой (`float` или `double`) в C++ может быть нетривиальной задачей из-за особенностей их представления и операций с ними. Числа с плавающей точкой могут содержать ошибки округления, поэтому простое сравнение с использованием оператора `==` может быть ненадежным.

Основные принципы проверки равенства

1. Использование эпсилон-значения:

  • Эпсилон-значение (epsilon) представляет собой небольшую величину, которая определяет допустимую разницу между двумя числами с плавающей точкой. Два числа считаются равными, если их абсолютная разница меньше эпсилон-значения.

2. Абсолютная и относительная разница:

  • Абсолютная разница может быть достаточной для чисел вблизи нуля.
  • Относительная разница лучше подходит для чисел, которые могут быть как большими, так и малыми.

Рассмотрим пример функции для проверки равенства двух чисел с плавающей точкой:

```cpp
#include <cmath>
#include <iostream>

// Функция для проверки равенства двух чисел с плавающей точкой
bool almostEqual(float a, float b, float epsilon = 1e-5f) {
    return std::fabs(a - b) < epsilon;
}

int main() {
    float x = 0.1f * 7.0f; // x должно быть примерно равно 0.7
    float y = 0.7f;

    if (almostEqual(x, y)) {
        std::cout << "x и y примерно равны." << std::endl;
    } else {
        std::cout << "x и y не равны." << std::endl;
    }

    return 0;
}
```

Объяснение

1. Параметры функции `almostEqual`:

  • `a` и `b`: два числа с плавающей точкой, которые нужно сравнить.
  • `epsilon`: допустимая разница между числами. Значение по умолчанию составляет `1e-5f`, что обычно достаточно для большинства приложений.

2. Использование `std::fabs`:

  • Функция `std::fabs` возвращает абсолютное значение разницы между `a` и `b`.

3. Сравнение с эпсилон-значением:

  • Если абсолютная разница меньше эпсилон-значения, то числа считаются равными.

Улучшенная версия с относительной ошибкой

Иногда для чисел, которые могут быть большими или очень малыми, имеет смысл использовать относительное сравнение:

```cpp
#include <cmath>
#include <iostream>

// Функция для проверки равенства двух чисел с плавающей точкой
bool almostEqualRelative(float a, float b, float epsilon = 1e-5f) {
    float diff = std::fabs(a - b);
    a = std::fabs(a);
    b = std::fabs(b);
    float largest = (b > a) ? b : a;

    return diff <= largest * epsilon;
}

int main() {
    float x = 1000.0f * 1.00001f; // x должно быть примерно равно 1000.01
    float y = 1000.01f;

    if (almostEqualRelative(x, y)) {
        std::cout << "x и y примерно равны." << std::endl;
    } else {
        std::cout << "x и y не равны." << std::endl;
    }

    return 0;
}
```

Объяснение

1. Вычисление абсолютной разницы:

  • Разница между `a` и `b` сохраняется в переменной `diff`.

2. Нахождение большего из двух чисел:

  • Определяется большее из двух чисел `a` и `b`, чтобы использовать его для нормализации разницы.

3. Относительное сравнение:

  • Разница `diff` сравнивается с произведением большего из чисел на эпсилон-значение. Это позволяет учитывать масштаб чисел.

Проверка равенства чисел с плавающей точкой требует использования эпсилон-значения для учёта возможных ошибок округления. Абсолютная и относительная разница помогают определить, насколько близки два числа. Эти методы обеспечивают более надёжное сравнение, чем простое использование оператора `==`.
